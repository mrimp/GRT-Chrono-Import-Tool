Extended metrics

Extended metrics – quick explainer

  • ±95% SD band – Mean ± 1.96 × SD. Roughly where about 95% of shots should land if your velocity is behaving nicely. Good for visualizing your “normal” speed window.
  • MAD (Median Absolute Deviation) – A robust version of spread. Like SD but much less sensitive to that one weird slow/fast shot. Lower MAD = more consistent.
  • CV% (Coefficient of Variation) – SD ÷ mean × 100. Lets you compare consistency between loads with different average speeds (e.g., 2800 vs 3000 ft/s).
  • ESD (ES/4) – Rule-of-thumb “estimated SD” from extreme spread: SD ≈ ES ÷ 4 for typical 5–10 shot strings. Good sanity check: if SD and ES/4 wildly disagree, something odd is going on.

Tip: these metrics don’t replace SD/ES — they just give you more ways to spot stable nodes and weird outliers.

✅ How to use in GRT

  1. Download the generated .grtload file.
  2. Open GRT and go to File → Open.
  3. Standard mode: use the Imported Measurement tab.
  4. Ladder mode: use the Imported Ladder tab; original ladder is unchanged.
  5. If used, view Imported Pictures and Imported Notes tabs.
  6. Check charge notes for AVG-E / SD / ES / MIN / MAX / UNIT.
  7. Check individual shots for PF and Time notes when present in the chrono file.

How to use GRT Chrono Importer

This is a quick start for your first run. You can ignore advanced options until you’re comfortable.

  1. Step 1 – Load base .grtload (optional)
    Open an existing GRT project if you want imported data to land inside it. If you skip this, the importer creates a fresh GRT file.
  2. Step 2 – Pick an import mode
    Standard: chrono files → one “Imported Measurement” tab.
    Ladder: chrono files per charge → one “Imported Ladder” tab.
    Sessions: Garmin multi-session workbook → “Imported Sessions” tabs (Garmin only).
  3. Step 3 – Load chrono files
    Drop your CSV/TSV/Excel files into the file area. Check that units and device match (Garmin/Athlon).
  4. Step 4 – Export to GRT
    Click Generate GRT file, then open the downloaded .grtload in GRT. Look for “Imported Measurement”, “Imported Ladder”, “Imported Sessions”, and optional “Chrono vs Sim” in the measurement tabs.

Not ready for the advanced stuff?
Leave Extended metrics and Chrono vs Sim OFF at first. Standard mode + SD/ES is plenty to get started.
